----------------------------------------------------
Dependencies (python 3.6)

1. install dlib
    pip install dlib-19.5.1-cp36-cp36m-win_amd64.whl
2. install the other packages to be needed.

----------------------------------------------------

Training model

1. In the folder 'train_images', add the folders with each person name and copy the person's images into it.
    1.1. each image should contain faces of same person.
    1.2. copy some face images into the folder 'test_images'
2. run 'trainng.py' to train the model.
3. run 'testing.pt' to test the current model.
